# Capa B · Ciencia de datos aplicada

*(Opcional.)* Para quien quedó enganchado y quiere **modelar y predecir**. Aquí entrenas modelos de
Machine Learning sobre **datos reales de compras públicas**, evaluándolos con honestidad.

!!! info "Un paso hacia un rol nuevo"
    A diferencia de la Capa A (mejorar en tu puesto), la Capa B ya es un paso hacia un perfil más
    técnico. Es opcional y exige más horas.

*Duración realista: ~6 semanas · 6–8 h/semana. Requisito: Capa A.*
