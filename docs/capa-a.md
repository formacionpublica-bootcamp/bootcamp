# Capa A · Datos sin miedo

**El corazón del bootcamp, para todos.** Al terminar, podrás responder preguntas con datos reales,
limpiar tablas sin sufrir, consultar con SQL, hacer un gráfico honesto y usar IA con criterio.

*Duración realista: ~6 semanas · 4–5 h/semana.*
