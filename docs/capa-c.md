# Capa C · IA aplicada al Estado

*(Opcional.)* Para quien quiere **construir con IA generativa**: resolver tareas de texto con LLMs,
darle a un modelo tus propios documentos (RAG) y encadenar todo en agentes.

!!! note "Requisito recomendado: A7"
    Esta capa se apoya en **[A7 · IA generativa y LLMs](capa-a.md)** (Capa A). Si no lo hiciste,
    empieza por ahí.

*Duración realista: ~5 semanas · 6–8 h/semana.*

!!! tip "▶ Pruébalo en el navegador (sin instalar, sin cuenta)"
    Abre los módulos en el [entorno interactivo](https://formacionpublica.cl/lite/lab/index.html): corre
    Python en tu navegador con **los datos ya incluidos**. Los notebooks de **profundización** son 100%
    offline; las lecciones que usan un LLM en vivo (C2/C3) conviene hacerlas en **Google Colab** (botones
    abajo).
