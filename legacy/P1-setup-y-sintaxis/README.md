Primer módulo del **Bootcamp de Datos para Funcionarios Públicos — Formación Pública**.
Módulo compartido (base para las líneas de Data Analytics y Data Scientist). Modalidad autoguiada, se ejecuta en Google Colab.

> Reemplaza `formacionpublica-bootcamp/bootcamp` en el enlace de arriba por la ruta real del repositorio cuando lo publiques en GitHub.

## ¿Qué vas a aprender?

A escribir y ejecutar un programa de Python básico: variables, tipos de datos, operadores, condicionales y salida de texto con f-strings. No necesitas saber nada de programación para empezar. Todos los ejemplos y ejercicios usan datos reales del gasto del Estado de Chile en compras públicas (ChileCompra).

**Competencia de salida:** escribir un pequeño programa que use variables, operadores, condicionales y salida de texto para responder una pregunta sobre datos públicos.

## Cómo empezar

1. Haz clic en el botón **Open in Colab** de arriba.
2. En Colab, ve ejecutando las celdas en orden (clic en la celda y **Shift + Enter**).
3. Al llegar a los ejercicios, completa las celdas con `# TODO` y ejecuta la **celda de chequeo** que sigue: verás ✅ si está correcto, o un mensaje que te dice qué revisar.
4. Terminaste cuando las cuatro celdas de chequeo muestran ✅.
