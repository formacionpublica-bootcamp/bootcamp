# M1 · Colecciones y bucles

Segundo módulo del **Bootcamp de Datos para Funcionarios Públicos — Formación Pública**.
Módulo compartido. Modalidad autoguiada, se ejecuta en Google Colab.

[![Open in Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/formacionpublica-bootcamp/bootcamp/blob/main/P2-colecciones-y-bucles/leccion.ipynb)

> Reemplaza `formacionpublica-bootcamp/bootcamp` por la ruta real del repositorio al publicar.

## ¿Qué vas a aprender?
A guardar muchos datos juntos (listas, tuplas, diccionarios) y recorrerlos con bucles (`for`, comprensiones) para resumirlos. Usa la serie real de compras públicas de Chile (ChileCompra), 2021–2025.

**Competencia de salida:** crear y recorrer colecciones de datos para resumir un conjunto de registros públicos.

## Cómo empezar
1. Abre `leccion.ipynb` con el botón **Open in Colab**.
2. Ejecuta las celdas en orden (Shift + Enter).
3. Completa los `# TODO` y corre la celda de chequeo: ✅ o pista de qué revisar.
4. Terminas cuando las cuatro celdas de chequeo muestran ✅.

## Contenido
- Listas: crear, indexar, `len`, `append`.
- Tuplas (datos fijos).
- Diccionarios: pares clave-valor; un registro como diccionario.
- Bucle `for` y patrón acumulador.
- Recorrer una lista de diccionarios.
- Comprensiones de lista (filtrar y transformar).
- Errores típicos.
- 4 ejercicios auto-corregidos sobre la serie real de compras públicas.
